¿Cómo puedo saber cuáles juegos funcionan en las Chromebooks para Cloud gaming?

Las Chromebooks para cloud gaming admiten tres plataformas principales: NVIDIA GeForce NOW, Xbox Cloud Gaming y Amazon Luna. Cualquier juego en la nube que funcione en esas plataformas será compatible con las Chromebooks para cloud gaming. Entre ellos, se incluyen Fortnite, The Witcher 3, Halo Infinite y más.

¿Funcionan otras Chromebooks para Cloud gaming?

Puedes acceder a las plataformas de cloud gaming en todas las Chromebooks. Las Chromebooks para cloud gaming están diseñadas para llevar tu experiencia de juego al siguiente nivel. Para ello, cuentan con funciones como el RGB, un teclado gaming con anti-ghosting, una frecuencia de actualización de 120 Hz y una pantalla de alta resolución.

¿Los Chromebooks para gaming en la nube son aptos para trabajar?

Los Chromebooks para gaming en la nube cuentan con la tecnología de los procesadores Intel Core y con 8 GB o más de RAM, por lo que son aptos para el trabajo y para hacer varias tareas a la vez en cualquier momento. También admiten aplicaciones como Google Workspace, Microsoft Office, Zoom y más, lo que te ayudará a trabajar de forma más eficiente.
